# User Management System API

## Description

The **User Management System API** is a lightweight backend application built using **Node.js core modules without any external frameworks like Express**. It provides a simple REST-style API that allows users to retrieve and manage user data stored in a JSON file.

The application demonstrates how to build a backend server using Node.js' built-in **HTTP, File System (fs), and Path modules**, while also implementing features such as filtering, searching, and sorting of user data.

This project is designed to help developers understand **how APIs work under the hood before using frameworks**.

---

## Summary

This project implements a **User Management API** that allows clients to retrieve and manipulate user data through HTTP requests.

Users can:

- Retrieve all users
- Filter users by gender
- Filter users by age
- Search users by name, city, or country
- Sort users by different fields

The project uses **JSON as a lightweight database** and demonstrates clean modular code organization.

---

## Features

- Retrieve all users
- Filter users by gender
- Filter users by minimum age
- Search users by:
  - country
  - city
  - user name
- Sort users dynamically
- Case-insensitive searching
- JSON file-based data storage
- Modular utility functions
- Error handling for invalid data
